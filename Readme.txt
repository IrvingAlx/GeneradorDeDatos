// COMO QUIERO QUE SE VEA //

							Generador de datos para bases de datos 

Datos que se pueden generar:

			A) Nombres
			B) Apellido Paterno
			C) Apellido Materno
			D) Telefonos
			E) Codigos Postales
			F) Trabajos 

Ingrese la cantidad de registros a generar 

Ingrese los datos que desea generar, presione S para avanzar

// QUE TENDRIA QUE HACER //

Primero muestra menu 

Segundo pregunta cuantos registros desea crear 

Tercero entra a un ciclo while y pregunta por los datos que usara 

Cuarto se guardara en un arreglo ademas de usar un contador para ver cuantos registros usara 

Quinto si es que el contador alcanza el numero max de datos se mostrara que ya no se pueden agregar mas letras 

Sexto con los datos el programa entrara a "datos.txt"
